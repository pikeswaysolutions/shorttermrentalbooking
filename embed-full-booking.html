Customer Booking Widget

This widget allows customers to browse properties and submit booking requests. Perfect for public-facing pages.

<!-- Property Booking Widget -->
<div id="property-booking-widget"></div>
<script>
  (function() {
    const iframe = document.createElement('iframe');
    iframe.src = window.location.origin + '/booking';
    iframe.style.width = '100%';
    iframe.style.height = '700px';
    iframe.style.border = 'none';
    iframe.style.borderRadius = '8px';
    iframe.style.overflow = 'hidden';
    document.getElementById('property-booking-widget').appendChild(iframe);
  })();
</script>

Customization Options

Height: Recommended minimum 700px to accommodate the booking wizard steps

Responsive: Width is set to 100% for mobile compatibility

Admin Calendar Widget

This widget displays the full rental calendar with booking management capabilities. Ideal for admin dashboards and internal tools.

<!-- Rental Calendar Widget -->
<div id="rental-calendar-widget"></div>
<script>
  (function() {
    const iframe = document.createElement('iframe');
    iframe.src = window.location.origin + '/admin/calendar';
    iframe.style.width = '100%';
    iframe.style.height = '900px';
    iframe.style.border = 'none';
    iframe.style.borderRadius = '8px';
    iframe.style.overflow = 'hidden';
    document.getElementById('rental-calendar-widget').appendChild(iframe);
  })();
</script>

Features

✅ View all bookings in a monthly calendar view

✅ Click dates to view details, approve, decline, or edit bookings

✅ Block dates directly from the calendar

✅ Color-coded status indicators (pending, confirmed, cancelled)
