Booking Calendar Widget - Embed Code

How to Embed the Calendar

Copy and paste this code into your admin dashboard or any webpage where you want the calendar to appear:

<!-- Rental Calendar Widget -->
<div id="rental-calendar-widget"></div>
<script>
  (function() {
    const iframe = document.createElement('iframe');
    iframe.src = window.location.origin + '/admin/calendar';
    iframe.style.width = '100%';
    iframe.style.height = '800px';
    iframe.style.border = 'none';
    iframe.style.borderRadius = '8px';
    iframe.style.overflow = 'hidden';
    document.getElementById('rental-calendar-widget').appendChild(iframe);
  })();
</script>

Customization Options

Height: Change iframe.style.height to adjust widget height (e.g., '1000px', '100vh')

Width: Set to '100%' for responsive or specify exact pixels (e.g., '1200px')

Border Radius: Adjust iframe.style.borderRadius for rounded corners
